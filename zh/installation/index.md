# Cocos2d-x 多平台的环境搭建

## 运行环境

1. 操作系统: macOS 10.13/Windows 7/Ubuntu 16.04
1. 引擎: cocos2d-x v3.16
1. IDE: Android Studio 2.3.3/Visual Studio 2015/XCode 8.3
