# Cocos2d-x v3.16 用户手册

欢迎使用 Cocos2d-x 用户手册，本手册包含引擎的介绍，功能组件的使用方法以及引擎在多个平台的环境搭建。能够帮助您快速上手 Cocos2d-x ，并开始进行跨平台游戏的开发。
